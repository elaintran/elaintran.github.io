Word Guess Game

The Word Guess Game is a vanilla JavaScript application that allows users to guess characters from the Super Mario franchise. Once a letter has been inputted from the keyboard, the application checks if it matches the letters of the selected word and updates the interface accordingly.

User Stories

One of UT Austin Coding Boot Camp's JavaScript projects is to create a word guess game with the following user stories:

  • I can listen to the keys pressed and reveal correctly guessed letters.
  • I can log number of wins, guesses remaining, and letters already guessed.
  • I can reset the game after a win or loss.
