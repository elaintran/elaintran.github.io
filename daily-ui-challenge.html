Daily UI Challenge

Daily UI is a challenge that sends a user interface prompt via email every day, over a course of a hundred days, to interpret and design. This challenge made me realize that not all design aspects are practical when it comes to real world application, and that functionality comes first and foremost, but I thoroughly enjoyed exploring new design styles, expanding my knowledge on programs, and getting a glimpse of how to handle certain scenarios in the future.
